# Subset NotoColorEmoji

Subset NotoColorEmoji font for Android T Egg

# Usage

### Install requirements
```shell
pip install -r requirements.txt
```
### Run script
```shell
python ./subset_noto_color_emoji.py
```

Output: `../src/main/assets/NotoColorEmojiSubset.ttf`
