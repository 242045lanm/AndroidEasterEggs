# Android Easter Eggs

整理了Android系统各正式版的彩蛋

[下载](https://www.pgyer.com/eggs)

![icon](./app/src/main/ic_launcher-playstore.png)

项目包含了系统彩蛋完整代码，旨在对系统彩蛋的整理和兼容，以保证大多数设备可以体验到不同版本的彩蛋，不会对系统彩蛋代码做过多修改。部分版本使用了系统新特性，低版本只能使用部分功能。
